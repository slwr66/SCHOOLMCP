# Changelog

Все значимые изменения в проекте документируются в этом файле.

## [0.3.3] - 2025-12-12

### Добавлено
- Health check endpoints (`/health`, `/tools`, `/`) для диагностики сервера
- Валидация конфигурации при запуске сервера
- Улучшенная диагностика регистрации инструментов
- Docker HEALTHCHECK для автоматического мониторинга
- Обработка ошибок с подробным логированием
- Подавление deprecation warnings для совместимости

### Исправлено
- Исправлена проблема с `asyncio.get_event_loop()` deprecation warning
- Исправлена проблема с `sse_app()` deprecation warning
- Улучшена обработка ошибок при создании приложения
- Исправлена документация по troubleshooting

### Изменено
- Обновлена документация в README.md
- Добавлены инструкции по настройке Container App на Cloud.ru
- Обновлена версия во всех файлах (Dockerfile, pyproject.toml, server.py)

### Документация
- Добавлен раздел про Health Check Endpoints в README
- Добавлен раздел про troubleshooting для Container App
- Обновлена инструкция по деплою в DEPLOYMENT.md
